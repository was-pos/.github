# WAS POS

Building modern, multi-tenant Point-of-Sale (POS) software — from cloud-native backend services to responsive front-end experiences.

## What We're Building

A distributed, scalable POS platform designed around:
- **Multi-tenant architecture** — DB-per-tenant model for secure, isolated SaaS deployments
- **API Gateway orchestration** — tenant-aware routing, JWT/OAuth2 authentication, gRPC-based interservice communication
- **Micro frontends** — modular, reusable UI components for fast feature rollout
- **Cloud-native infrastructure** — Kubernetes-based deployments with CI/CD automation

## Tech Stack

`Java` · `Spring Boot` · `Spring Cloud Gateway` · `Next.js` · `React` · `Kubernetes` · `Docker` · `MariaDB`
